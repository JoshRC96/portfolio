LED Office Status Indicator

The "office-status-indicator" project is a desktop application created with Python to control an LED, effectively indicating the status of an office as either "in use" (RED) or "available" (WHITE). Comprising two key components, the project's user interface is designed in "userApp.py" using Tkinter, facilitating easy status changes. This script communicates seamlessly with an Arduino board through serial port connections, allowing real-time control of the LED. The second component, "ledControl.ino," manages the Arduino UNO R3 board and governs the RGB LED's color based on signals from "userApp.py."

  • Python
  • C++ (Arduino)
  • Elego UNO R3 Board
